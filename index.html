Welcome to The Silent Net

A work in progress.

The Silent Net (otherwise known as the sNet) is a decentralized, distributed network that uses a chain-based algorithm to allow the registration, the modification and the access of free domains via a fast and efficient database system.

The result is a peer-to-peer network that provides fast connection to the domain you wish to visit.
